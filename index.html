Project: mscgen_js

A faithful reimplementation of mscgen in javascript that allows you to embed sequence charts in html. There's also an online interpreter that runs in the browser itself.

Project: WordyWordy

A web app that presents text one word at a time. You can drop any text on it and it will start reading it to you. It also remembers where you left off.

I started this to test some stuff for mscgen_js, but it ended up being useful in itself.
