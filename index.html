Side projects

Writing software is still fun, though, so I do that in my spare time.

  • mscgen.js - a re-implementation of MscGen that makes it available on the web
  • Atom editor plugins for writing and previewing MscGen and Graphviz
  • state machine cat - a language and accompanying tooling for writing beautiful state charts.
